## Overview

**Finder** is a web application designed to make playing badminton easier and more social.  
Players can find partners, discover nearby courts with real-time distance display, and book courts with integrated **ZaloPay** payments.  
Court owners can manage their courts, handle user posts, and track revenue through an intuitive dashboard.

## ‚ú® Key Features

### For Players

-   Login with **Google** or **Facebook**
-   Create and manage posts to find badminton partners
-   View and search posts from other players
-   Find badminton courts nearby with **real distance calculation**
-   Book courts and pay securely through **ZaloPay**
-   Edit personal information and view booking/post history

### For Court Owners

-   Manage courts (add / edit / delete)
-   Auto-complete address input using **Goong Map API**
-   Approve or reject user posts
-   View revenue analytics by **day, month, year**, or per court
-   Export transaction data as **CSV files**
